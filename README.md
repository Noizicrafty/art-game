# Арт-игра

Веб-игра для учителя и детей с комнатами по коду, работает по сети через сервер.

## Запуск на сервере (порт 1919)

```bash
node server.js
```

Сервер стартует на `0.0.0.0:1919`.

- На этом же компьютере: `http://localhost:1919`
- Для детей в сети: `http://<IP_сервера>:1919`

## Игровой цикл

1. Учитель нажимает «Учитель», выбирает режим (один/несколько) и тему.
2. Создается комната с кодом, код показывается справа.
3. Дети с любых браузеров в сети вводят имя и код комнаты.
4. В лобби учитель видит список подключившихся детей и может исключать.
5. Учитель запускает игру.
6. 2 минуты: запомнить эталонный рисунок.
7. 5 минут: рисование на canvas (paint UI) или загрузка фото.
8. Несколько детей: голосование и таблица 1/2/3 места.
9. Один ребенок: показ эталона рядом с рисунком и поздравление.
