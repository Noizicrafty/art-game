# Арт-игра

Простое веб-приложение для игры взрослого с детьми в формат «запомни и нарисуй».

## Запуск

Откройте `index.html` в браузере.

## Игровой цикл

1. Взрослый выбирает формат: один ребенок или несколько.
2. Взрослый выбирает тему (картинки из ассетов репозитория).
3. Создается лобби и код комнаты.
4. Дети вводят имя/фамилию и код комнаты.
5. Взрослый видит список детей и может исключать их из лобби.
6. После старта раунда дети 2 минуты запоминают выбранный рисунок.
7. Затем 5 минут рисуют в встроенном paint-интерфейсе или загружают фото.
8. Если игра на нескольких:
   - дети голосуют за лучший рисунок;
   - показываются 1, 2 и 3 место в экране победителей.
9. Если игра на одного:
   - показывается оригинал и рисунок ребенка рядом;
   - выводится поздравление ребенку.
